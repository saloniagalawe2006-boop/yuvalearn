# Design Report

## Layout Strategy
Maine wireframe ko header, hero section, content cards, aur footer me divide kiya...

## Design Decisions
- Colors: dark header/footer for contrast, light background for readability
- Font: Arial for clean sans-serif look
- Spacing: consistent padding/margin for visual balance

## Responsiveness
Media queries add ki 768px breakpoint par, taaki mobile par nav aur cards stack ho jayein.

## Challenges Faced
- Cards ko equally align karne me flexbox ka use kiya, initially grid try kiya tha lekin flex zyada simple laga
- Navigation responsive banane me thoda time laga
